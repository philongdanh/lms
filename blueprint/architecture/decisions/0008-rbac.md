# ADR-008: RBAC with 5 Seed Roles

## Context

Cần authorization model linh hoạt.

## Decision

**RBAC** với 5 roles: `root-admin`, `tenant-admin`, `teacher`, `parent`, `student`.

## Rationale

Permission-based, hierarchy rõ ràng, dễ mở rộng.
