# 0024: HeroUI

Cần thư viện UI Component đẹp, hiện đại và accessibility tốt.

---

## Decision

Sử dụng **HeroUI** (trước đây là NextUI).

---

## Rationale

- **Built on Tailwind**: Tương thích hoàn toàn với TailwindCSS.
- **Aesthetics**: Thiết kế đẹp, hiện đại mặc định.
- **Accessibility**: Tuân thủ WAI-ARIA.
- **React Server Components**: Hỗ trợ tốt Next.js App Router (client components
  where needed).
