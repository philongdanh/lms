# ADR-017: Recharts

Cần thư viện biểu đồ cho Dashboard thống kê (học tập, doanh thu).

---

## Decision

Sử dụng **Recharts**.

---

## Rationale

- **React Component Based**: Dễ dàng sử dụng như các component React thông
  thường.
- **D3-based**: Mạnh mẽ nhưng API đơn giản hơn D3 trần.
- **Responsive**: Hỗ trợ responsive tốt.
- **Customizable**: Dễ dàng custom tooltip, legend, axis.
