# 24: RetroUI

Thư viện ui component đẹp, hiện đại với phong cách neobrutalism độc đáo

---

## Decision

**[retroui](https://www.retroui.dev/)**

---

## Rationale

- **NeoBrutalism Design**: Phong cách thiết kế độc đáo, hiện đại và nổi bật
- **Built on Tailwind**: Tương thích hoàn toàn với TailwindCSS
- **Built-in Charts**: Hỗ trợ sẵn các biểu đồ, không cần thư viện chart riêng
- **React Components**: Dễ dàng sử dụng như các component React thông thường
- **Lightweight**: Nhẹ hơn so với các UI library khác
