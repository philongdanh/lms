# Roadmap

Lộ trình phát triển sản phẩm.

---

## Phase 1: MVP

| Sprint      | Thời gian     | Trọng tâm    | Bàn giao                                          |
| :---------- | :------------ | :----------- | :------------------------------------------------ |
| **S1**      | 20/01 - 02/02 | Foundation   | Monorepo, Database, CI/CD.                        |
| **S2**      | 03/02 - 16/02 | Auth & Users | RBAC, User Profile, Sessions.                     |
| **S3**      | 17/02 - 02/03 | Learning     | Lesson Structure, Quiz Engine, Progress Tracking. |
| **S4**      | 03/03 - 15/03 | Polish       | Gamification (Basic), Bug fixes, UAT.             |
| **Release** | 16/03 - 22/03 | **Go live**  | Feedback, Cloud Deploy.                           |

**Scope:**

- **Core:** Auth, Phân quyền (RBAC).
- **Learning:** Cấu trúc bài học, Quiz Engine, Theo dõi tiến độ.
- **Engage:** Gamification cơ bản (XP, Level, Badges).

---
