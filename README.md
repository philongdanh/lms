# LMS Documentation

Tài liệu chính thức cho hệ thống Learning Management System (LMS).

---

## Target Audience

| Vai trò       | Tài liệu          | Mục đích                               |
| :------------ | :---------------- | :------------------------------------- |
| **PO**        | Blueprint         | Định hướng sản phẩm, roadmap, backlog  |
| **SA**        | Blueprint + Spec  | Thiết kế hệ thống, tech stack, modules |
| **Dev**       | Spec + Onboarding | Triển khai API, UI, codebase           |
| **QA**        | QA                | Test cases, automation, benchmarks     |
| **AI Agents** | Spec + QA         | Đọc spec để sinh code, test            |
| **End User**  | Handbook          | Hướng dẫn sử dụng sản phẩm             |
