**Phiên bản đầu tiên của tài liệu LMS!**

## Các tính năng (Features)

### Tài liệu (Docs)

- **User Guide**: Hướng dẫn cho người dùng cuối hệ thống LMS
  - Hướng dẫn Bắt đầu (Getting Started)
  - Phần câu hỏi thường gặp (FAQ)
- **Developer Guide**: Tài liệu kỹ thuật cho lập trình viên
  - Hướng dẫn cài đặt (Setup)
  - Quy trình phát triển (Development workflow)
  - Hướng dẫn triển khai (Deployment guide)
  - Quy tắc đóng góp (Contributing guidelines)

### Thông số kỹ thuật (Specs)

- **Business**: Tầm nhìn, lộ trình, chỉ số, ràng buộc, thuật ngữ
- **Architecture**: Thiết kế hệ thống, danh sách công nghệ (tech stack), mô hình
  dữ liệu (data model), tài liệu quyết định kiến trúc (ADRs)
- **Modules**: Auth, Content, Learning, Admin, Analytics, Gamification,
  Tournament, Realtime
- **Cross-cutting**: API Gateway, Security, Performance, Monitoring, Error
  Handling
- **Quality**: Chiến lược test, tự động hóa, benchmarks

### Hệ thống thiết kế (Design)

- **Research**: Personas, hành trình người dùng (User journeys), kiểm thử khả
  năng sử dụng (Usability testing)
- **Design System**: Các hướng dẫn, Design tokens
- **Wireframes**: Bản vẽ phác thảo UI/UX

### Tài liệu API (API)

- **OpenAPI**: Thông số kỹ thuật API (main.yaml)
- **Contracts**: Các hợp đồng API
- **Mocks**: Dữ liệu giả định và các ví dụ
